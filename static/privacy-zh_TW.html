1. 我們收集哪些個人數據及如何使用

我們提供由獨立開發者創建的多樣化在線 H5 遊戲。您可以在不註冊或登錄的情況下訪問我們的服務。網站上的某些遊戲由第三方控制，包括位於 EU/EEA 以外的第三方，這些第三方將根據其自身的隱私政策處理數據。對於我們主辦的遊戲，我們僅在必要的最小範圍內收集和處理您的個人數據，以實現以下目的，並根據適當的法律依據進行處理。

1.1

為了提供一些基於位置的服務，例如與附近的人組隊，我們可能會收集您的 IP 地址、GPS、RFID 和您的家庭地址。

這種處理活動基於您的同意。我們將在您啟用此功能時要求您的同意，您有權隨時撤回您的同意。

1.2

為了改善我們的服務並修正您在遊戲中遇到的錯誤或問題，我們可能會收集相關信息，包括但不限於您的遊戲行為數據、設備信息和網絡狀態。

這種處理活動基於我們的合法利益，不會影響您的法律權利和利益。

1.3

為了遵守適用的法律（例如反洗錢規定），我們可能會收集您的設備和網絡信息。

這種處理活動對於遵守我們所需遵守的法律義務是必要的。

2. Cookie 和類似技術

我們使用 Cookie 和其他類似技術（例如 Web 燈塔、日誌文件、腳本和 eTags）（以下稱“Cookie”）來提升您使用服務的體驗。Cookie 是小型文件，當其放置在您的設備上時，可以幫助我們提供某些功能和特性。

您可以選擇允許安裝這些 Cookie，或隨後禁用它們。您可以接受所有 Cookie，或指示設備或 Web 瀏覽器（如適用）在安裝 Cookie 時提供通知，或通過調整設備中的相關 Cookie 保留功能來拒絕接受所有 Cookie。然而，如果您拒絕安裝 Cookie，遊戲可能無法按設計運行。
